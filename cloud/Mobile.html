Mobile

For the convenience of users, we have created a mobile version of the Timesheet Reports and Gadgets plugin. Now you can use Timesheet app from phone.

Install

Below is the installation guide for the mobile version (designed for mobile devices and made as Progressive Web Application, see link for information about supported platforms):

Visit Timesheet Reports and Gadgets mobile app page on your mobile device at https://timesheet-plugin.herokuapp.com/m and authorise it with your Atlassian account.

Click Settings in the upper right corner of the screen.

Select Add to Home screen.

Add a shortcut to the main screen of the phone.

To start the application, tap the shortcut on the phone screen.

Usage

In mobile version there are Time Entry and Time Reports views available. Time Entry - single day view, provides convenient way to record working time, i.e. start, stop, resume timer, log work done or edit previous records.
