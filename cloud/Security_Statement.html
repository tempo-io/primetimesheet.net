Security Statement

Effective as of April 24, 2020.

Prime Timesheet s.r.o. cares about the protection of your data.

Our Atlassian Connect Add On (Prime Timesheet) processes your data by (1) uploading your time tracking data from an Atlassian Jira Cloud instance, processing it on the client side (in an end-user’s internet browser), and sending it back to the Cloud instance; or (2) instructing the Jira Cloud instance to process it.

Data Security

Atlassian is responsible for protecting your data in the Atlassian Jira Cloud. For information on how Atlassian protects your data, see Atlassian’s Trust page.

Our Add On protects your data during transit by implementing HTTPS using TLS.
Prime Timesheet s.r.o. is certified as a Cloud Security Compliant vendor with Atlassian, see here.

Managing Security Vulnerabilities

Security vulnerability bugs, when found, get highest priority and are fixed in 2 days and rolled out immediately, without any notification.
