Roadmap

For the long time Timesheet plugin evolves based on your feedback and feature requests, and I also have some features in mind. Please vote for any or many enhancements below, publicly or at corresponding bitbucket issue, if you'd like it to be implemented.

Holidays Calendar (Issue #297)

Curently there is basic support for holidays (see Issue #257), however it could be better if holidays were retrieved automatically from corporate calendar, e.g. Confluence Team Calendars or any other ical web service, e.g. http://www.calendarlabs.com/ical-calendar-holidays.php

Please like it publicly or vote for bitbucket Issue #297, if you'd like this feature to be implemented.

 

Configurable weekends (Issue #14)

The functionality of visualizing Saturday and Sunday is not perfect. There are some countries where different days are non-working days (e.g. Friday and Saturday in Israel). Maybe it would reasonable to make plugin configurable by selecting in configuration dialog in a multi-selection list the days which are to be marked as weekends.

Please like it publicly or vote for bitbucket Issue #14, if you'd like this feature to be implemented.

 

Sum Tasks/Stories into Epic (Issue #360)

Summing sub tasks (Issue #301) is nice, though it could be extended it to sum worked hours into GreenHopper Epic issues.

Please like it publicly or vote for bitbucket Issue #360, if you'd like this feature to be implemented.

Email Timesheet Report (Issue #316)

Since version 2.5 it is possible to Subscribe to Time Sheet Report to receive it regularly by email. See more details here. It may be even better way to keep an eye on worklogs of your team, than Timesheet Gadget on Dashboard.

Do you like it too? Let all know!
