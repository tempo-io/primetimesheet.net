Extended JIRA time tracking means

License

You can purchase commercial license here or request demonstration license below to enable new features, like Timesheet Configuration, Timesheet Holidays, Timesheet Email and Worked Time Chart in JIRA Cloud. You don't need license unless you need these features.

Note, this is temporary solution. It has no variation by number of users in JIRA, so price is the same for all size companies. And only payments via PayPal are supported with no any kind of invoice beside standard PayPal confirmation are possible.

Once you pay chosen amount, you will shortly receive license code to email address you use for PayPal payment. Copy paste it into https://yourjirahost/secure/LicenseConfiguration!default.jspa (edit yourjirahost), though normally you should have come here from that page.

Please, specify name of your Organisation (will be displayed in License) and click Pay Now below to be redirected to payment page. Note, it's not possible get license for longer period, because Timesheet Reports and Gadgets add-ons will be replaced with Time Reports Add-On for JIRA Cloud in near future. Should you have any questions, please do not hesitate to write an email.
