Frequent Questions

If you have not found your answer here or in Atlassian Answers, do not hesitate to write me an email in English, Russian or Czech language at your convenience.

Hours logged per user per day?

You just need to Group by: Worked User.

Where is Time Reports menu?

Time Reports report is accessible via menu Projects - some project - Reports tab - Other category.

It can be also open from dashboard using arrow icon in top right corner of the Time Reports gadget.

Additionally you can add quick link with Application Navigator - it allows to set its links to be viewable just by the specified groups in a really straight-forward manner.

Timesheet Report missing?

It's replaced with Time Reports, please see announcment for more details.

Missing features?

Collapse Field Groups option - should be added soon.

Holidays is likely to be added later.

Support for Timesheet Email may not be added soon. It requires significant rework, and storing worklogs server-side, that is out of add-on goals.

REST endpoint for raw worklogs - please use JIRA native REST api, as described here.
